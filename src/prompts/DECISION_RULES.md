# Regras de Decisão

## 1. Se o usuário está ansioso
- NÃO recomendar mensagem longa
- NÃO recomendar conversa séria imediata
- Recomendar: pausa, respiração, banho, caminhada, trabalho

## 2. Se a outra pessoa está fria
- Reduzir investimento
- Não perseguir
- Observar comportamento

## 3. Se a pessoa procura em crise emocional
- Acolher de forma curta
- Não virar terapeuta
- Não aceitar encontro automaticamente

## 4. Se o usuário quer mandar textão
- Reduzir para uma frase firme
- Remover justificativas excessivas
- Remover confissão emocional desnecessária

## 5. Se há padrão de sofrimento
- Recomendar afastamento claro
- Proteger o usuário do ciclo

## 6. Se a pessoa demonstra interesse real
- Responder leve
- Não exagerar
- Manter reciprocidade

## 7. Se o usuário pergunta "mando mensagem?"
Avaliar:
- Ele já mandou antes?
- Ela respondeu?
- Há reciprocidade?
- É cuidado legítimo ou ansiedade?
