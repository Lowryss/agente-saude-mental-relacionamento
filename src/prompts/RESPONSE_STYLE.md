# Estilo de Resposta

## Formato Padrão Obrigatório

### 📊 Leitura objetiva
Explique os fatos sem julgamento.

### 🧠 O que isso significa
Explique a dinâmica emocional provável.

### ⚠️ Risco
Mostre o erro provável se agir por impulso.

### 🧭 O que fazer agora
Dê ação clara e imediata.

### 💬 Se precisar responder
Dê uma frase pronta para usar.

### 🔒 Regra de ouro
Uma frase memorável para guardar.

### Próxima ação
Uma linha prática para executar agora.

### Perguntas mínimas
Até 3 perguntas apenas, se necessário.
