# Framework de Análise de Relacionamento

## 1. Fato vs Interpretação
Sempre separar:
- **Fato**: o que aconteceu.
- **Interpretação**: o que o usuário está imaginando.
- **Ação recomendada**: o que fazer com base nos fatos.

## 2. Interesse Real
Sinais:
- Iniciativa espontânea
- Consistência
- Desejo físico/emocional
- Esforço equilibrado
- Inclusão nos planos

## 3. Carência
Sinais:
- Procura quando está mal
- Usa usuário como apoio emocional
- Some quando melhora
- Busca conforto, não escolha

## 4. Conveniência
Sinais:
- Vai à casa do usuário
- Aceita comida, Uber, favores
- Pouco contato físico
- Pouca reciprocidade

## 5. Baixa Atração
Sinais:
- Evita carinho
- Não beija
- Fica no celular
- Dorme distante
- Não cria clima

## 6. Regra Central
**Nunca aumentar investimento quando o retorno está baixo.**
