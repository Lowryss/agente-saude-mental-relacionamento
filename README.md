# Agente de IA para Relacionamentos (Low Ticket)

Este projeto tem como objetivo criar um agente de inteligência artificial especializado em aconselhamento de relacionamentos, focado em oferecer orientação comportamental e emocional de forma empática, inspirada em experiências de sucesso com o ChatGPT.

## 🎯 Visão do Produto
O produto será uma solução "low ticket" com um design extremamente atraente e uma experiência de usuário (UX) fluida, permitindo que os usuários recebam conselhos práticos sobre como lidar com conflitos, comunicação e dinâmica de casal.

## 🚀 Pilares do Projeto
1.  **Empatia e Personalidade:** Uma IA que não apenas responde, mas valida sentimentos e propõe ações concretas.
2.  **Design Irresistível:** Interface moderna, acolhedora e minimalista (estética mental health/wellness).
3.  **Acessibilidade:** Modelo de baixo custo com entrega de alto valor percebido.

## 🛠️ Plano de Execução

### Fase 1: Definição e Personagem (Prompt Engineering)
- [ ] Definir o "tom de voz" (acolhedor, direto, analítico ou empático).
- [ ] Criar o System Prompt base com técnicas de Chain-of-Thought para análise de conflitos.
- [ ] Estabelecer limites éticos e de segurança (saúde mental).

### Fase 2: Estrutura Técnica
- [ ] Backend: FastAPI ou Node.js para integração com APIs de LLM (OpenAI/Anthropic).
- [ ] Frontend: Next.js ou React com Tailwind CSS para um design "Premium".
- [ ] Banco de Dados: Memória de curto prazo para conversas contextuais (Redis ou Supabase).

### Fase 3: Design e UI/UX
- [ ] Paleta de cores suave (tons pastéis, azul suave, verde menta).
- [ ] Tipografia elegante.
- [ ] Animações fluidas para simular "pensamento" e empatia.

### Fase 4: Monetização e Lançamento
- [ ] Estrutura de pagamento (Stripe/Mercado Pago).
- [ ] Landing Page de alta conversão.
